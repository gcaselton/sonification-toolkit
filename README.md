# Data Sonification Toolkit

An application to play sounds of astronomical objects using the Strauss package.

## Pre-requisites

- React v19.1.0
- Vite v6.1.1
- Node v20.9.0
- npm v10.2.2
- FastAPI v0.0.7
- Python v3.9.0

## Application structure

The backend uses fastapi and the frontend uses React + Vite

- backend/
  - api/
    - __init__.py
    - light_curve.py
  main.py
- frontend/
  - public
  - src/
    - assets/
    - components/
    App.css
    App.tsx
    index.css
    main.tsx
    vite-env.d.ts
    .gitignore
